| Felt (dark) | Deep green | `#1e3d1a` |
| Points A | Crimson | `#8b1a1a` |
| Points B | Gold | `#c5a028` |
| Player 1 checker | Ivory | `#faf5e8` / `#e8dcc8` / `#c9b896` |
| Player 2 checker | Obsidian | `#4a4a4a` / `#2a2a2a` / `#111111` |
| Accent / text | Gold | `#c5a028` / `#e8c840` |
| Locked glow P1 | Gold | `#c5a028` |
| Locked glow P2 | Silver | `#a0a0b0` |
| Dice face | Parchment | `#f5f0e0` / `#d9d0b8` |
| Dice pips | Dark brown | `#2a1508` |
