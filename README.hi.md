भौतिकी-आधारित बोर्ड गेम टूलकिट। एक नियतात्मक 2डी भौतिकी इंजन जिसमें SVG रेंडरिंग है, एक पूर्ण रणनीति बोर्ड गेम (सीज गैमन), और एनिमेशन को डीबग करने के लिए क्रोम डेवलपर टूल्स एक्सटेंशन - ये सभी एक टर्बोरेपो मोनोरेपो में मौजूद हैं।

## पैकेज

| पैकेज | विवरण | स्थिति |
| --------- | ------------- | -------- |
| **@mcptoolshop/physics-svg** | SVG और कैनवस रेंडरर के साथ एक नियतात्मक 2डी भौतिकी इंजन। | npm पर प्रकाशित। |
| **@mcp-tool-shop/siege-types** | भौतिकी, गेम स्टेट और एनिमेशन के लिए साझा टाइपस्क्रिप्ट प्रकार। | आंतरिक। |

## एप्लिकेशन

| एप्लिकेशन | विवरण |
| ----- | ------------- |
| **SiegeGammon** | एक इनवर्स-बैकगामन रणनीति गेम - टुकड़ों को तैनात करें, आगे बढ़ें और उन्हें सीज ज़ोन में लॉक करें। |
| **Anim DevTools** | भौतिकी सिमुलेशन और एनिमेशन टाइमलाइन की जांच के लिए क्रोम एक्सटेंशन। |

## सीज गैमन

सीज गैमन, बैकगामन को एक नए तरीके से पेश करता है: पारंपरिक तरीके से टुकड़ों को हटाने के बजाय, खिलाड़ी रिजर्व से टुकड़ों को खाली बोर्ड पर तैनात करते हैं और सभी 15 टुकड़ों को प्रतिद्वंद्वी के होम क्षेत्र में लॉक करने की दौड़ लगाते हैं। बोर्ड का स्थानिक उलटा - आपका गार्rison उनके सीज ज़ोन को ओवरलैप करता है - शुरुआत से अंत तक संपर्क और तनाव को बनाए रखता है।

## भौतिकी इंजन

एक निश्चित 60 हर्ट्ज़ टाइमस्टेप पर सेमी-इम्प्लिसिट यूलर इंटीग्रेशन, उसी दृष्टिकोण का उपयोग बॉक्स2डी और रैपियर द्वारा किया जाता है। सर्कल, एएबीबी और एसएटी टक्कर का पता लगाना और आवेग-आधारित समाधान। बाधाओं को प्रोजेक्टेड गौस-सीडल विधि से हल किया जाता है। स्थिर होने पर बॉडी निष्क्रिय हो जाते हैं। SVG रेंडरिंग अनिवार्य ट्रांसफॉर्म अपडेट का उपयोग करता है (शून्य रिएक्ट री-रेंडर); 200 से अधिक बॉडी होने पर कैनवस बैकअप सक्रिय हो जाता है।

## तकनीकी स्टैक

- **TypeScript 5.7** / **Node 22+** / **pnpm 10**
- ऐप्स और लाइब्रेरी बिल्ड के लिए **Vite 6**
- मोनोरेपो ऑर्केस्ट्रेशन के लिए **Turbo 2.4**
- परीक्षण के लिए **Vitest 3**
- गेम ऐप में **React 19** + **GSAP 3**

## शुरुआत कैसे करें

```bash
pnpm install
pnpm dev
```
